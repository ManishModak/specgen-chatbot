## üé® Coding Standards

### **Next.js & React**
*   **App Router:** Use Server Components by default. Add `'use client'` only when interactivity (state, hooks) is needed.
*   **Vercel AI SDK:** Use `useChat` and `streamText` for fluid, streaming responses.
    *   Model: **Gemini 1.5 Pro** (for reasoning capabilities).
*   **TypeScript:** Strict mode is on. Define interfaces for all Props.
    *   *Avoid `any`.*

### **UI/UX: "Gamer Aesthetic"**
*   **Framework:** Tailwind CSS.
*   **Theme:** Dark mode by default.
*   **Colors:** High contrast. Black/Dark Gray backgrounds with Neon accents (Green/Purple/Blue).
*   **Typography:** Clean, sans-serif (Inter or similar). Monospace for specs/code.

### **Data & State**
*   **RAG (Retrieval Augmented Generation):**
    *   The chatbot does not just "guess". It must query `data/embeddings.json` (or vector DB) to retrieve real product specs before answering.
    *   Implement a "Tool" or "Function Call" for the LLM to search the product catalog.
